## 1. Описание продукта

### Что это

«Сова» — персональный финансовый помощник на базе искусственного интеллекта. Основной интерфейс — бот в Telegram, дополненный веб-дашбордом для детальной аналитики. Продукт объединяет три направления в одном месте: учёт личных финансов, управление инвестиционным портфелем и персонализированную AI-аналитику.

Принципиальное отличие от существующих решений: большинство финансовых приложений хорошо фиксируют транзакции, но не помогают пользователю понять, что делать с этими данными. «Сова» — не трекер расходов, а финансовый аналитик, который анализирует, объясняет и подсказывает.

### Для кого

Русскоязычные пользователи с любым уровнем финансовой грамотности — от людей, которые впервые задумались о бюджете, до активных инвесторов. Искусственный интеллект автоматически адаптирует стиль общения и глубину подачи информации под уровень пользователя.

### Ключевые возможности

- **Учёт финансов** — быстрый ввод расходов естественным языком (например, «кофе 350»), распознавание фото чеков, автоматическая категоризация
- **Интеграции** — автоматическая синхронизация данных из ZenMoney (личные финансы) и Т-Банк Инвестиции (портфель)
- **AI-аналитика** — ответы на вопросы вроде «Куда уходят деньги?», «Что с моими финансами?», анализ портфеля, моделирование сценариев накоплений
- **Управление инвестициями** — просмотр портфеля, отслеживание P&L, совершение сделок прямо из Telegram
- **Умные уведомления** — алерты о крупных тратах, приближении к лимитам бюджета, просадках портфеля, дивидендах, персонализированных новостях рынка
- **Дайджесты** — ежедневные и еженедельные финансовые сводки с AI-инсайтами
- **Веб-дашборд** — графики, аналитика расходов, портфель по секторам, прогресс по финансовым целям

### Бизнес-модель

Freemium с оплатой за AI-функции по модели pay-per-use:

- **Бесплатно:** учёт финансов, интеграции, автокатегоризация, базовые запросы, уведомления
- **Платно (AI-баланс):** глубокая аналитика (5 руб.), анализ портфеля (8 руб.), аналитика по тикеру (10 руб.), генерация дайджеста (12 руб.), свободный AI-чат (3 руб. за сообщение)

Оплата через Telegram Stars. Пользователь пополняет баланс (от 100 руб.), каждый AI-запрос списывает фиксированную сумму. Баланс бессрочный, возможен вывод неиспользованных средств.

### Платформа и технологии

Telegram-бот как основная точка входа. Веб-дашборд (PWA, mobile-first) для глубокой аналитики. Авторизация — через Telegram Login Widget, без паролей. AI работает на базе Claude API (Anthropic).

### Юридические ограничения

Продукт предоставляет аналитику и образовательный контент, но не даёт директивных инвестиционных рекомендаций (в соответствии со ст. 6.1 N 39-ФЗ). Торговые функции отделены от AI-ответов. Соответствие 152-ФЗ о персональных данных, база данных размещена на серверах в РФ.

## 3. Вопросы для исследования

### 3.1. Объём рынка

- Оценка TAM/SAM/SOM для рынка персональных финансовых помощников в России на 2026-2028 гг.
- Объём рынка приложений для учёта личных финансов (в деньгах и пользователях)
- Объём рынка розничных инвестиционных сервисов (не брокеры, а надстройки: агрегаторы, аналитика, робоэдвайзеры)
- Объём рынка AI-ассистентов в финансовом секторе (B2C-сегмент)
- Динамика роста каждого из сегментов за последние 3 года и прогноз
- Количество пользователей Telegram в РФ, которые используют финансовые боты

### 3.2. Конкурентный ландшафт

- Полный перечень прямых конкурентов: приложения для учёта финансов (ZenMoney, CoinKeeper, Monefy, Дребеденьги и др.)
- Косвенные конкуренты: встроенная аналитика банковских приложений (Т-Банк, Сбер, Альфа), брокерские приложения с аналитикой
- AI-конкуренты: финансовые чат-боты, AI-надстройки над банками
- По каждому конкуренту: ключевые функции, ценообразование, размер аудитории, сильные и слабые стороны, отзывы пользователей
- Наличие Telegram-ботов для финансов (учёт, инвестиции, новости) — кто, сколько пользователей, функции
- Незакрытые потребности пользователей, которые не покрывают текущие решения

### 3.3. Целевая аудитория

- Демографический профиль пользователей финансовых приложений в РФ: возраст, пол, доход, город, образование
- Поведенческие сегменты: как люди ведут учёт финансов сегодня (вручную, в приложениях, в Excel, никак)
- Какой процент ведёт бюджет регулярно, какой пробовал и бросил, какой никогда не пробовал
- Основные болевые точки: что не устраивает в текущих решениях
- Отношение к AI в финансовых сервисах: уровень доверия, опасения, ожидания
- Какая доля аудитории готова совершать инвестиционные операции через бот (не через приложение брокера)
- Пересечение аудиторий: сколько людей одновременно пользуются и приложениями учёта финансов, и брокерскими сервисами

### 3.4. Ценообразование и готовность платить

- Willingness to pay за AI-функции финансового помощника (в рублях за запрос и в рублях в месяц)
- Восприятие модели pay-per-use (оплата за запрос) vs. подписка (фиксированная ежемесячная плата) — что предпочтительнее для целевой аудитории
- Оптимальный ценовой диапазон за один AI-запрос (текущая гипотеза: 3-12 руб.)
- Средний ожидаемый месячный расход на AI-функции
- Ценовая эластичность: при какой цене пользователи откажутся от сервиса
- Порог пополнения: минимальная комфортная сумма для первого пополнения (гипотеза: 100 руб.)
- Готовность платить за отдельные функции: аналитика расходов, анализ портфеля, AI-чат, дайджесты

### 3.5. Каналы привлечения пользователей

- Где целевая аудитория узнаёт о новых финансовых продуктах: каналы информирования
- Оценка CAC (стоимость привлечения клиента) по каналам: Telegram Ads, контент-маркетинг, SEO, партнёрства с финансовыми блогерами, YouTube, рефералки
- Какие Telegram-каналы и сообщества популярны у целевой аудитории (личные финансы, инвестиции)
- Эффективность вирусных механик (реферальные программы, бонусы за приглашение) в сегменте финтех
- Роль сарафанного радио и отзывов в принятии решения о подключении финансового сервиса

### 3.6. Позиционирование и дифференциация

- Как пользователи воспринимают существующие решения: ZenMoney, CoinKeeper, встроенную аналитику банков
- Какие ценностные предложения резонируют с аудиторией: «экономия времени», «понимание своих финансов», «AI-советник», «всё в одном месте», «бот в Telegram»
- Что является главным мотиватором к использованию финансового помощника и что — барьером
- Как воспринимается концепция «не просто трекер, а советник» — насколько это убедительно
- Ассоциации и восприятие бренда «Сова» (мудрость, спокойствие, доверие) в контексте финансового продукта

### 3.7. Telegram как платформа для финансового продукта

- Размер и динамика аудитории Telegram в РФ (MAU, DAU)
- Текущая аудитория финансовых ботов в Telegram: категории, размеры, активность
- Уровень доверия пользователей к Telegram-ботам при работе с финансовыми данными
- Барьеры доверия: что мешает людям передавать финансовую информацию боту
- Сравнение восприятия «бот в Telegram» vs. «отдельное приложение» для финансовых задач
- Опыт оплаты через Telegram Stars: знают ли пользователи о такой возможности, пользовались ли, уровень доверия

### 3.8. Барьеры и риски

- Что мешает людям начать пользоваться приложениями для учёта финансов (лень, сложность, непонимание ценности, страх за данные)
- Почему люди бросают использование финансовых трекеров: retention-проблемы
- Страхи, связанные с предоставлением доступа к банковским данным через API
- Отношение к передаче финансовых данных в AI-систему (особенно зарубежную — Anthropic/Claude)
- Страхи, связанные с совершением сделок через бот (а не через приложение брокера)

### 3.9. Приоритизация функций для MVP

- Какие функции привлекают больше всего: учёт бюджета, AI-аналитика расходов, управление инвестициями, умные уведомления, дайджесты, торговля через бота
- Рейтинг функций по привлекательности и готовности платить (матрица «важность / готовность платить»)
- Минимальный набор функций, за который пользователи готовы зарегистрироваться
- Какие функции являются «must have» для удержания, а какие — «nice to have»
- Интерес к распознаванию чеков по фото vs. ручной ввод vs. автоматическая синхронизация

## 4. Желаемый формат результатов

### Итоговый отчёт должен включать:

1. **Аналитическая записка (Executive Summary)** — 3-5 страниц с ключевыми выводами, цифрами и рекомендациями для принятия стратегических решений

2. **Полный отчёт** — развёрнутый документ по каждому блоку вопросов с данными, графиками, таблицами и источниками

3. **Конкурентная карта** — визуальная матрица конкурентов по осям «функциональность / цена», с указанием аудитории и ключевых особенностей каждого

4. **Портреты целевых сегментов** — 3-5 детализированных профилей аудитории (персоны) с описанием демографии, поведения, мотивации, болей и готовности платить

5. **Ценовая модель** — обоснованные рекомендации по ценообразованию на основе данных о willingness to pay, с расчётом оптимальной цены за запрос и среднего ARPU

6. **Приоритизация функций** — ранжированный список функций по привлекательности для аудитории, с разбивкой по сегментам

7. **Рекомендации по GTM-стратегии** — каналы привлечения с оценкой CAC, рекомендации по позиционированию и ключевым сообщениям

8. **Приложения** — методология исследования, опросные материалы, сырые данные, список источников

### Форматы файлов:

- Отчёт: PDF + редактируемая версия (DOCX или Google Docs)
- Данные и таблицы: Excel / Google Sheets
- Презентация с ключевыми выводами: PDF + PPTX
