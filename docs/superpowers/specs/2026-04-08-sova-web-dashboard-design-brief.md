# Дизайн-бриф: Веб-дашборд "Сова"

**Дата:** 2026-04-08
**Версия:** 1.0

---

## 1. Общее описание продукта и контекст

### Что за продукт

"Сова" — персональный AI-финансовый аналитик, объединяющий учёт расходов, управление инвестиционным портфелем и персонализированную AI-аналитику. Позиционирование: не трекер расходов, а **советник**.

### Бизнес-модель

Freemium с pay-per-use для AI-функций:
- **Бесплатно:** учёт финансов, интеграции, автокатегоризация, уведомления
- **Платно (AI-баланс):** глубокая аналитика (5-12 руб/запрос), AI-чат (3 руб/сообщение)

LTV/CAC = 4.36, ARPU = 150 руб/мес.

### Роль веб-дашборда

Вторичный интерфейс. Основной — Telegram-бот. Дашборд для того, что неудобно в боте: графики, таблицы, drill-down аналитика. Доступен из бота (кнопка) или напрямую по URL. Авторизация через Telegram Login Widget.

### Целевая аудитория

**P1 "Осознанный оптимизатор"** (28-38 лет, 80-150K руб/мес) — ведёт бюджет, инвестирует, хочет понимать паттерны трат.

**P2 "Начинающий инвестор"** (25-35 лет) — открыл брокерский, купил 3-5 акций, не понимает как оценить портфель.

**P3 "Семейный бюджетчик"** (30-45 лет) — хочет видеть "куда ушли деньги".

## 2. User Flows

### 2.1 Первый вход

Пользователь → URL → лендинг с лого Совы + кнопка "Войти через Telegram" → Telegram Login Widget → подтверждение → JWT (24ч) → главная страница.

### 2.2 Первый визит (нет данных)

Все виджеты в empty state → баннер "Подключи ZenMoney или T-Bank" → кнопки подключения → OAuth flow → данные появляются.

### 2.3 Просмотр аналитики

Навигация → "Аналитика" → текущий месяц → donut расходов по категориям (клик = drill-down) → bar chart доходы vs расходы → heatmap трат → топ категорий с трендами.

### 2.4 Управление портфелем

Навигация → "Портфель" → общая стоимость + P&L → pie chart по секторам/валютам/типам → таблица позиций (сортируемая) → клик на позицию = боковая панель с деталями → line chart стоимости → дивидендный календарь.

### 2.5 Работа с целями

Навигация → "Цели" → карточки с прогресс-барами → "+ Новая цель" → форма → прогноз достижения → AI-рекомендация (платно).

### 2.6 Настройки

Вкладки: Интеграции (подключить/отключить ZM/T-Bank) → Уведомления (toggles с порогами) → Профиль (уровень, валюта, часовой пояс) → AI-баланс (пополнить/вывести/история) → Категории (CRUD).

### 2.7 Пополнение AI-баланса

Виджет в header → "Пополнить" → выбор суммы [100/300/500/другая] → deep link в Telegram-бота для оплаты через Stars → история операций.

### 2.8 Новости

Навигация → "Новости" → курсы валют + индексы сверху → персонализированная лента → каждая новость с пометкой влияния на портфель → фильтр: все / мой портфель / по сектору.

## 3. Детальное описание страниц

### 3.1 Экран входа

- Логотип "Сова" (иконка совы + название)
- Кнопка "Войти через Telegram"
- 1 предложение о продукте
- Ссылки: Политика конфиденциальности, Условия использования
- Минималистичный, тёмный фон, акцентная кнопка

### 3.2 Главная (Overview)

| Виджет | Данные | Приоритет |
|--------|--------|-----------|
| Общий баланс | Сумма по всем счетам | 1 |
| Мини-график доходы/расходы | Line/area chart за 30 дней | 2 |
| Стоимость портфеля | Сумма + P&L за день | 3 |
| Прогресс по целям | 1-3 цели с прогресс-барами | 4 |
| AI-инсайт дня | 1 бесплатный инсайт | 5 |
| Последние 5 транзакций | Список | 6 |

Layout: 2 колонки на десктопе, 1 колонка на мобильном.

### 3.3 Транзакции

- Фильтры: период, категория, счёт, поиск
- Группировка: по дням / неделям / месяцам
- Таблица: дата, описание, категория (с иконкой), счёт, сумма
- Массовое редактирование категорий
- Экспорт в CSV
- Мобильный: карточки вместо таблицы

### 3.4 Аналитика

| Виджет | Тип визуализации |
|--------|-------------------|
| Расходы по категориям | Donut chart (кликабельный drill-down) |
| Тренд по категориям | Stacked bar chart |
| Доходы vs расходы | Bar chart (двойной) |
| Кешфлоу | Calendar heatmap |
| Топ категорий | Таблица с мини-sparkline |
| Аномалии | Карточки-алерты |

Переключатель периода: месяц / квартал / год / произвольный.

### 3.5 Портфель

| Виджет | Тип визуализации |
|--------|-------------------|
| Общая стоимость + P&L | Число + badge |
| Распределение | Pie chart (по секторам/валютам/типам — tabs) |
| Таблица позиций | Тикер, кол-во, ср.цена, текущая, P&L, доля % |
| История стоимости | Line chart |
| Дивидендный календарь | Timeline / list |
| Операции | Таблица |

Клик по позиции → боковая панель с деталями.

### 3.6 Цели

- Карточки с прогресс-барами
- "+ Новая цель" → форма (название, сумма, дедлайн, счёт)
- Прогноз достижения
- AI-рекомендация (платно)
- Сетка: 2 колонки десктоп, 1 мобильный

### 3.7 Новости и рынок

- Мини-виджеты: USD, EUR, CNY, MOEX, S&P 500
- Персонализированная лента
- Пометка влияния на портфель
- Фильтр: все / мой портфель / по сектору

### 3.8 Настройки

Вкладки:
1. **Интеграции** — карточки ZM/T-Bank, статус, подключение/отключение
2. **Уведомления** — toggles с настройкой порогов
3. **Профиль** — уровень, валюта, часовой пояс
4. **AI-баланс** — баланс, история, пополнение (deep link в бот), вывод
5. **Категории** — CRUD пользовательских категорий

## 4. Требования к дизайну

### 4.1 Общий подход

- **Mobile-first (PWA)**: 375px → 768px → 1280px+
- Touch-targets ≥ 44x44px
- **Dark theme как основной** (light theme — позже)

### 4.2 Цветовая палитра

| Роль | Цвет | Применение |
|------|-------|------------|
| Primary | Amber/Gold (#F5A623) | CTA, акценты, иконка совы |
| Background | Dark (#0D1117) | Основной фон |
| Card | Dark Blue (#1C2333) | Фон карточек |
| Text Primary | Light (#F0F6FC) | Основной текст |
| Text Secondary | Gray (#8B949E) | Вторичный текст |
| Success | Green (#3FB950) | Доход, рост, достигнутое |
| Error | Red (#F85149) | Расход, просадка, ошибки |
| Warning | Orange (#D29922) | Предупреждения |
| Info | Blue (#58A6FF) | Информация, ссылки |

Amber/Gold как primary — ассоциация с совой, дифференциация от зелёного финтеха.

### 4.3 Типографика

- **Шрифт:** Inter (заголовки + текст)
- **Цифры:** tabular (monospace) figures для выравнивания
- Минимальный размер: 14px десктоп, 16px мобильный
- Межстрочный: 1.5 текст, 1.2 заголовки

### 4.4 Графики

Все графики:
- Интерактивные (hover = tooltip)
- Анимированные при появлении (+ prefers-reduced-motion)
- Адаптивные
- В цветовой палитре продукта
- Библиотека: Recharts или Nivo

### 4.5 Адаптивность

| Breakpoint | Layout | Навигация |
|------------|--------|-----------|
| Mobile (320-767px) | 1 колонка, карточки вместо таблиц | Bottom tab bar |
| Tablet (768-1023px) | 2 колонки, таблицы с h-scroll | Side nav (collapsible) |
| Desktop (1024px+) | 2-3 колонки, полные таблицы | Side nav (expanded) |

### 4.6 Навигация

**Мобильная (bottom tab bar):**
```
[Главная] [Аналитика] [Портфель] [Новости] [Ещё]
```

**Десктопная (left sidebar):**
```
[Логотип Сова]
Главная
Транзакции
Аналитика
Портфель
Цели
Новости
---
Настройки
[AI-баланс: 42.50 руб] [Пополнить]
[Аватар] [Имя]
```

### 4.7 Компоненты дизайн-системы

- **Card** — #1C2333, border-radius 12px, padding 16-24px, subtle border
- **Button** — primary (amber), secondary (outline), ghost, disabled; sm/md/lg
- **Input** — тёмный фон, светлый border, focus с primary
- **Badge** — для +/- значений (success/error)
- **Toggle** — для настроек
- **Progress bar** — для целей
- **Table** — zebra-striping, sortable headers
- **Chart tooltip** — тёмный с акцентным border
- **Empty state** — иллюстрация совы + текст + CTA
- **Skeleton loader** — анимированные placeholder

### 4.8 Иконки

- Lucide Icons (линейные, 24px, stroke 1.5-2px)
- ~20 иконок для категорий расходов
- Маскот "Сова" — экран входа, empty states, AI-инсайты

## 6. Технические ограничения

- React 18 + TypeScript, PWA
- Авторизация: Telegram Login Widget (нельзя кастомизировать)
- JWT 24 часа — нужен UX для re-login
- Пополнение баланса через deep link в Telegram (не веб-форма)
- REST polling (60с портфель, 5м транзакции), нет WebSocket
- WCAG 2.1 AA (контрастность 4.5:1)
- Юридическое: ссылки на политику + согласие ПД + кнопка удаления аккаунта

## Примечания для дизайнера

1. Дашборд = аналитика и визуализация. Не дублировать бот (ввод расходов, торговля, AI-чат).
2. Маскот "Сова" — ненавязчиво: экран входа, empty states, AI-инсайты.
3. Финансовые числа — центральный элемент: крупные, читаемые, разделители тысяч, цветовая индикация +/-.
4. Аудитория 28-38 лет, Telegram-пользователи. Современный дизайн, не "интернет-банк 2015 года".
